# 🛡️ Security Hardening Skill

Use this skill when auditing for security vulnerabilities or when a specific threat (e.g., SQL injection, secret leakage) is detected.

## Instructions

1. **Threat Analysis:** Explain the potential impact of the vulnerability (e.g., "Hardcoded API key can lead to unauthorized access").
2. **Mitigation Strategy:** Provide the industry-standard fix (e.g., using environment variables, parameterized queries, or encryption).
3. **Defense in Depth:** Suggest additional layers of security (e.g., rotating keys, adding WAF rules).
